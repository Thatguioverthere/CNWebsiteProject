Outcome 2.3

Assess the strengths and weaknesses of your final project

There is definitely a lot of changes that could be made to improve the website. Had I been given more time, I would have endeavoured to make it look more professional, or more like a blog. However, I can happy with its current look and feel. I attempted to make it 'dark mode' so it strains less on the eyes, which should make for a pleasant viewing experience.

Outcome 2.4

Describe further improvements you can make to your project

One Further improvement I could make is to rework the whole layout of the website, I could also add more images to break up the text and make it read more like a blog. Another improvement would be to change up the colour scheme again. Adding on from this idea, I could add the elements for allowing the user to swap between light and dark mode, rather than setting full base colours that the user is forced to use.
